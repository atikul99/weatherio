A React-based weather searching application that provides real-time weather information by city name.
Built to demonstrate API integration, search functionality, and clean UI development.

## Features

- Search weather by city name
- Current temperature & weather condition
- Humidity & wind speed
- Real-time data from API
- Fast & responsive UI

## Tech Stack

- React.js
- JavaScript (ES6+)
- Weather API (WeatherAPI)

```bash
